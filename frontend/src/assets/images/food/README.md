# Food Images Directory

This directory contains all food images used throughout the application.

## Directory Structure:
- `/dishes/` - Individual dish images
- `/categories/` - Category/cuisine type images  
- `/restaurants/` - Restaurant images
- `/icons/` - Small food icons for UI elements

## Image Specifications:
- Dishes: 400x300px, optimized JPG/PNG
- Categories: 200x200px, square format
- Restaurant headers: 600x300px, banner format
- Icons: 64x64px, PNG with transparency

## Naming Convention:
- Use kebab-case (lowercase with hyphens)
- Include cuisine type where relevant
- Example: `south-indian-meals.jpg`, `chicken-biryani.jpg`
